# OSUI toggle-button

### 基本属性

| 参数 | 说明 | 类型 | 默认值
| --- | --- | --- | --- |
| hasSelectNumber | 需要展示选项数值的样式 | boolean |  |
| hasShowNumber | 需要展示button数值的样式 | boolean |  |
| num | hasSelectNumber或者hasShowNumber为true,显示的数值 | number |  |
| className | 按钮的名称 | string |  |
| beforeIcon | 展示按钮前面的icon | React.ReactNode |
