# OSUI Space

## API

### 基本属性

支持所有antd属性：见[antd文档](https://ant.design/components/space-cn/#API)

### OSUI 调整

OSUI Space 增加了css variables 覆盖的能力可以通过修改 --space-small, --space-middle, --space-large 来调整space的大小。

small: 10px;
middle: 20px;
large: 30px;
