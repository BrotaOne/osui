# OSUI Button

2020-9-28 通过UE视觉验收

### 基本属性

支持所有antd属性：见[antd文档](https://ant.design/components/button-cn/#API)

### OSUI新增属性

| 参数 | 说明 | 类型 | 默认值
| --- | --- | --- | --- |
| type | 除antd原有属性以外，新增加strong值 | string | primary, ghost, dashed, link, text, default, strong |
| success | 成功按钮 | boolean、undefined | undefined |
| error | 失败按钮 | boolean、undefined | undefined |
| warning | 警告按钮 | boolean、undefined | undefined |
| flexCenter | button内容是否flex居中 | boolean、undefined | undefined |
