# OSUI Form

## API

### 基本属性

支持所有antd属性：见[antd文档](https://ant.design/components/form-cn/)

### OSUI新增属性

#### Form.Item

| 参数 | 说明 | 类型 | 默认值
| --- | --- | --- | --- |
| hint | 对所填内容的说明 | string | undefined |
| validateMessageLayout | 报错信息展示位置 | 'inline', 'default' | 'default' |

### 说明：
`hint`与`help`的区别： `help`会冲掉validate message，即当传入`help`之后，报错的时候会显示`help`内容而不是validate message的。这个不符合我们的设计，所以新增了hint属性。

`validateMessageLayout`有两种`inline`|`default`，`default`垂直排放，`inline`会在Item后面显示，`inline`没有经过完善的实际使用测试，有可能有样式问题。
