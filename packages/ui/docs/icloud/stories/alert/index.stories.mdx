# OSUI Alert

## API

### 基本属性

支持所有antd属性：见[antd文档](https://ant.design/components/alert-cn/)

### 新增属性

| 参数 | 说明 | 类型 | 默认值
| --- | --- | --- | --- |
| actions | message中增加的action，例如：查看文档连接 | ReactNode | - |
| expandable | 是否展开收起展示message，为true时只展示一行内容 | boolean | - |
| countDown | 倒计时自动关闭alert | number | -1 |

### 说明

**注意**：当时用`countDown`自动关闭的时候，`onClose`不再是antd close按钮的触发时间，是一个没有参数的回调函数
